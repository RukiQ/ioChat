## ☛ 介绍

#### 该版本为聊天室的升级版本，实现了 [socket.io官网](https://socket.io/get-started/chat/) 的课后练习中要求的部分功能：
  

- **监听用户进入(connected)或离开(disconnected)**

- **显示谁在线**


## ✍ 使用

	npm install
	npm start
